This is a polished Next.js starter tailored for a Lead Data Engineer / Solution Architect.

## Quickstart
1. Install Node.js LTS
2. `npm install`
3. `npm run dev`
4. Open http://localhost:3000

## Notes about extra packages
- This scaffold includes a Three.js interactive component (react-three/fiber + drei). During `npm install`, those packages will be pulled.
- ShadCN is referenced as a dependency for future polishing. We'll add concrete components if you want them.

## MDX / Blog
- Add .md files into `content/posts/` to publish blog posts.

## GPU Notebook
- Use RunPod or Colab for CUDA-enabled notebooks. For self-hosted iframe, set `NOTEBOOK_IFRAME` environment variable on Vercel.
